---
description: 端末内に分散している野良スキル（Agent Skills）を共通リポジトリへ安全に収集・同期するスキル
---

# 目的
ユーザーのローカル開発環境（PC内）に散らばっているユーザー作成のAIエージェントスキルを自動で検索・選別し、中央連携用の `skills` リポジトリへ収集・同期します。

# 実行手順

## 1. 探索対象の特定
以下のOSに応じた検索コマンドを実行し、PC内に存在する `skill` 名を冠するディレクトリ一覧を取得してください。

**Windows (PowerShell) の場合の検索コマンド:**
```powershell
Get-ChildItem -Path "$HOME" -Filter "*skill*" -Recurse -Directory -ErrorAction SilentlyContinue | Where-Object { $_.FullName -notmatch "AppData" } | Select-Object FullName
```

**Mac / Linux の場合の検索コマンド:**
```bash
find ~ -type d -name "*skill*" 2>/dev/null | grep -v "Library"
```

## 2. 収集対象の選別
出力されたディレクトリパスの中から、VSCodeの拡張機能システム内部などを除外し、ユーザーが個人的に記述したと思われる独自スキル（例: `.codex/skills/...`, `各プロジェクト/.agents/...` など）を選別します。
すでにリポジトリ内に存在しているスキルと重複がある場合はスキップします。

## 3. 機密情報のチェックとクレンジング（重要）
> [!WARNING]
> 共有リポジトリに野良スキルをコピーする前に、必ず内部（特に各スキルの `SKILL.md`）を読み、以下の機密情報が直書きされていないか確認・修正してください。

- **APIキー/トークン**: `sk-...` などの文字列があれば警告し、環境変数を利用するコードへ書き換える。
- **個人情報/固定パス**: 特定のローカル絶対パス（例: `C:\Users\username\...`）を使っている場合は、動的なパス取得や相対パスへ修正する。

## 4. 共有リポジトリへのコピー
安全が確保されたスキルフォルダを、共通リポジトリの `agent/` ディレクトリ下へコピーします。

## 5. GitHubへの保存・共有
コピーが終わったら、Gitコマンドを実行して収集したスキルを保存・同期してください。
```bash
git add .
git commit -m "feat: ローカルの未収集スキルを取り込み"
git push
```
最後に、今回収集できたスキルのリストと、もし機密情報の修正を行った場合はその詳細をユーザーに報告してタスクを完了します。
