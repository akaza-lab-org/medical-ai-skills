# Tobu Ticket Downloader Skill

このスキルは、東武ネット特急券（トブチケ）の購入履歴から領収書を自動的にダウンロードするためのものです。

## 構成

- `scripts/download_receipts.py`: メインのダウンローダー。Playwrightを使用します。
- `scripts/get_tobuchike_detail.py`: デバッグや詳細情報の抽出用。

## 前提条件

- `playwright` がインストールされていること。
- Chrome ブラウザが利用可能であること。

## 使い方（AIへの指示）

1. ユーザーから「トブチケの領収書をダウンロードして」と依頼されたら、`scripts/download_receipts.py` を実行する準備をします。
2. スクリプトは対話形式で設定（スキップ設定、期間指定）を求めます。
3. 実行中、ブラウザが起動しログインを求められます。ユーザーにログインと履歴一覧の表示を依頼し、準備ができたら Enter を押すよう案内してください。
4. ダウンロードされた PDF は、プロジェクトルートの `receipts/` ディレクトリに保存されます。

## 注意事項

- ログインパスワードなどの機密情報はスクリプトに含めないでください。
- ブラウザ操作は `--headless=False` で行われるため、ユーザーのデスクトップ上でブラウザが立ち上がります。
